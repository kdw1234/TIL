## 논리연산

AND:두개의 bool 값이 전부 true 일경우 결과값은 true 로 나온다
OR:두개중의 값이 하나라도 true 일경우 false 로 나온다 
NOT:값을 반전시켜준다
XOR:두개의 조건중에 하나만 참일 경우


Min:입력받은 값중 가장 작은값을 반환
